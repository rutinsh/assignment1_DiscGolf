Topic: Develop a website using HTML and CSS.
Choose a website topic based on your interests, experiences or hobbies.
Task 1 – Setup the project structure.
1.1. Create a new project folder named assignment1_<project_name>.
1.2. Inside this folder, create the following subfolders:
• css/ → Store CSS files.
• images/ → Store images and media files.
1.3. Inside the root folder, create the following HTML files:
• index.html (Homepage)
• about.html (About or Team page)
• services.html (or another relevant page based on the topic)
• contact.html (Contact page)
1.4. Create a CSS file inside the css/ folder:
• styles.css (Main stylesheet)
1.5. Link the styles.css file to all HTML files using the <link> tag inside the <head> section.
Task 2 - Develop the HTML Structure.
2.1. Homepage (index.html)
2.1.1. A navigation bar with links to other pages.
2.1.2. A hero section with:
o A heading
o A short description
o A call-to-action button
2.1.3. A featured section introducing the website’s purpose.
2.1.4. A three-column section highlighting important features.
2.1.5. A footer with social media links and copyright information.
2.2. About Page (about.html)
2.2.1. A team introduction section with:
o Member names and roles.
o Profile pictures
2.2.2. A mission statement or company description.
2.2.3. A timeline or history section using lists.
2.3. Services or Additional Page (services.html)
2.3.1. A list of services/products/features in <section> with:
o A title
o A description
o An icon or image
2.4. Contact Page (contact.html)
2.4.1. A contact form with:
o Full Name
o E-mail
o Message
o Submit button (<button type="submit">)
2.4.2. A list of contact details (i.e. email, phone, address, etc).
Task 3: Apply CSS for Styling
3.1. General Styling
3.1.1. Set a background color and font-family.
3.1.2. Define a global style for <h1>, <h2>, <p>, <a>, etc.
3.2.Navigation Bar Styling:
3.2.1. Set the background color of <nav>.
3.2.2. Style menu items (<ul>, <li>, <a>) with spacing and hover effects.
3.2.3. Use display: flex; for horizontal alignment.
3.3.Hero Section Styling
3.3.1. Add a background image or gradient.
3.3.2. Center text and buttons.
3.3.3. Style the button (<button>) with a hover effect.
3.4.Grid or Flexbox Layouts
3.4.1. Use display for sections.
3.4.2. Ensure equal spacing and alignment for images, text, and buttons.
3.5.Form Styling
3.5.1. Style input fields with padding, border-radius, and spacing.
3.5.2. Add a hover effect for the submit button.
3.6.Footer Styling
3.6.1. Set a dark background with white text.
3.6.2. Align text centrally.
Task 4: Implement Responsive Design
4.1. Use CSS Media Queries to adjust the layout for different screens:
4.1.1. Convert grid layouts into single-column layouts.
4.1.2. Reduce font sizes and button widths on smaller screens.
4.1.3. Set flex-direction.
4.2. Test responsiveness using:
Chrome DevTools (Inspect → Toggle Device Toolbar)
Different screen sizes (mobile, tablet, desktop)
